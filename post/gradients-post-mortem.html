First, weâ€™ll be introduced to the nabla operator, which is denoted by $\bar{\nabla}$. This is a vector operator. $$\bar{\nabla} = \hat{i} \frac{\partial}{\partial x} + \hat{j} \frac{\partial}{\partial y} + \hat{k} \frac{\partial}{\partial z}$$ An operator is something that acts on some quantity/function and makes some changes to it according to some certain rules. The nabla operator is constructed by taking the summation of all space derivative along the directional unit vectors. It has no value unless it is acted upon on some function.

The gradient is the multiplication of some scalar function to the operator, which is denoted by grad($\Psi$) or $\bar{\nabla}\Psi$. $$\bar{\nabla}\Psi = \hat{i} \frac{\partial \Psi}{\partial x} + \hat{j} \frac{\partial \Psi}{\partial y} + \hat{k} \frac{\partial \Psi}{\partial z}$$ As we can see the gradient of some scalar function is a vector quantity. It contains directional unit vectors.

These all are just mathematics. But what does gradient means physically? What happens when we take the gradient of some physical scalar quantity?

Such a physical scalar quantity is the temperature. Let us consider taking the gradient of the temperature ($\Psi$).

$\bar{\nabla}\Psi$ has two characteristics, magnitude, and direction. The magnitude is found by, $$\left | \bar{\nabla}\Psi \right | = \sqrt{(\frac{\partial \Psi}{\partial x})^{2}+(\frac{\partial \Psi}{\partial y})^{2}+(\frac{\partial \Psi}{\partial z})^{2}}$$ The magnitude gives the change of the scalar function as the space change. For a better view, let us consider one component of $\bar{\nabla}\Psi$ along the $x$-axis. $\hat{i} \frac{\partial \Psi}{\partial x}$ is the change of the scalar function with respect to $x$ along the $x$-axis.

The direction points towards the direction in which the function is increasing most quickly with distance. So, if we take the gradient of temperature in a room it will point towards the maximum value of temperature.

That is where the temperature is maximum. And, $\frac{1}{r^2}$ is the rate of change of temperature with distance.
